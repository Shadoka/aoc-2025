# aoc-2025
Advent of Code 2025 solutions

Unless otherwise specified you can always build the application by going into a directory and
running
```
go build
```

## Day 1 Part 1

Download the input data [(link here)](https://adventofcode.com/2025/day/1) and give it as input to the application.
```
./puzzle1 input.txt
```

## Day 1 Part 2

Input data is the same as in part 1. Build day1part2 and give the input to the application:
```
./puzzle2 input.txt
```

## Day 2 Part 1

Download the input data [(link here)](https://adventofcode.com/2025/day/2). Add an empty line at the bottom and pass it to
the application:
```
./puzzle3 input.txt
```

## Day 2 Part 2

Download the input data [(link here)](https://adventofcode.com/2025/day/2). Add an empty line at the bottom and pass it to
the application:
